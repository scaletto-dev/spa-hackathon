## 🎨 Path Aliases

Bạn có thể import dễ dàng hơn với path aliases:

```typescript
// Thay vì:
import { Header } from '../../../admin/components/layout/Header';

// Dùng:
import { Header } from '@admin/components/layout/Header';
```

Available aliases:

-   `@/*` - src/\*
-   `@admin/*` - src/admin/\*
-   `@client/*` - src/client/\*
-   `@shared/*` - src/shared/\*
